# Contributing

## Reporting Issues

If you encounter bugs or have questions:
1. Check existing [issues](https://github.com/SpaceTime-Vision-Robotics-Laboratory/nser-ibvs-drone/issues)
2. Open a new issue with:
    - Your environment (OS, Python version, GPU)
    - Steps to reproduce
    - Error messages/logs

## Code Contributions

We welcome pull requests for:
- Bug fixes
- Documentation improvements
- New features that align with the project goals

Before submitting:
1. Fork the repository
2. Create a branch (`git checkout -b feature/your-feature`)
3. Follow existing code style (we use [Ruff](https://github.com/astral-sh/ruff) for linting)
4. Test your changes
5. Submit a PR with a clear description

## Questions

For questions about the paper or methodology, feel free to open an issue or contact the authors.
